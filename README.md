## Installation

Clone down this repository. You will need ``` node```, ``` Sanity``` and ```npm``` installed globally on your machine.


Installation:
```bash
  npm install
```
To Run Test Suite:
```bash
  npm test
```
To Start Server:
```bash
  npm start
```
To Visit App:
```bash
  localhost:3000/
```

Navigate to backend folder and install Sanity:
```bash
  npm install -g @sanity/client
```
## Run Locally
